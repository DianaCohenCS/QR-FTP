# QR-FTP
Workshop on Ad Hoc Networks at Open University (20588-2021-summer): Optical File Transfer Protocol (QR-FTP), coding in Go on Linux using QR-codes and web-cams, connected to VMs.

Refer Documentation/Slides for overall presentation of the client-server project, where each Go program is mapped to a web-cam watching the other's screen exchanging messages as QR-codes.
